# 🛒 E-commerce Sales Analysis

## 📌 Project Overview
This project analyzes e-commerce sales data using Python, SQL, and Power BI to identify sales trends, profit performance, and customer insights.

## 🎯 Business Objective
- Analyze sales and profit
- Identify top-performing categories
- Find high-performing states
- Track monthly sales trends
- Build an interactive Power BI dashboard
